# NeuroDetective Diagnostic Center

## Description
An interactive neurological emergency simulation
where players analyze symptoms and select diagnostic
tests for possible stroke or brain protection failure.

## Technologies
- HTML
- CSS
- JavaScript

## Features
- Countdown emergency timer
- Interactive diagnostic pathways
- Stroke detection simulation
- Blood-brain barrier analysis
- Neural signal testing

## Neuroscience Concepts
- Neurovascular coupling
- Stroke physiology
- BBB protection
- Neural communication
